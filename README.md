# 🌍 Multilingual Sentiment Analyzer

An AI-powered web application that detects sentiment (Positive, Negative, Neutral) from text in multiple languages using pretrained transformer models.

---

## 🚀 Features

- 🌐 Supports multiple languages
- 🧠 AI-based sentiment analysis using Transformers
- 🔍 Automatic language detection
- 📊 Confidence score for predictions
- 🎨 Beautiful Streamlit UI with background styling
- ⚡ Fast real-time predictions

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎨 (Frontend UI)
- Flask ⚙️ (API backend)
- Hugging Face Transformers 🤗
- langdetect 🌐
- PyTorch 🔥
